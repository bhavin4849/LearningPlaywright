# LearningPlaywright

This repository contains JavaScript practice files and beginner exercises used for learning programming concepts and Playwright-related scripting.

## Contents

- Chapter-based JavaScript exercises in the `chapter_01` folder
- Small standalone scripts for arrays, conditionals, loops, and functions
- Practice examples for core JavaScript fundamentals

## Getting Started

1. Install Node.js on your machine.
2. Open the project folder in your terminal.
3. Run any script with:

```bash
node <filename>.js
```

This repository is intended for learning and experimentation.
